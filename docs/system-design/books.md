- System Design Interview Volume 1 & 2 by Alex xu

- Database Internals: A Deep Dive Into How Distributed Data Systems Work Book by Alex Petrov
- Crafting Interpreters Book by Robert Nystrom
- The Practice of Programming Book by Brian Kernighan and Rob Pike
- Programming Pearls Book by Jon Bentley
- Coders at WorkBook by Peter Seibel
- The Art of Computer Programming Book by Donald Knuth (Vol 1 & 2)
- Release It! Design and Deploy Production-Ready Software Book by Michael T. Nygard
- Fundamentals of Software Architecture: An Engineering Approach Book by Mark Richards and Neal Ford
- Building Microservices: Designing Fine-Grained Systems Book by Sam Newman
- "Site Reliability Engineering" by Google -> https://sre.google/books/
- A Philosophy of Software Design Book by John Ousterhout
- Domain-Driven Design: Tackling Complexity in the Heart of Software Book by Eric Evans
- Working Effectively with Legacy Code Book by Michael Feathers

- Code Complete Book by Steve McConnell
- Thinking in Systems: A Primer Book by Donella Meadows


### C books

- The C Programming Language. 2nd Edition Book by Brian Kernighan and Dennis Ritchie
- Operating Systems: Three Easy Pieces Book by Andrea Arpaci-Dusseau and Remzi Arpaci-Dusseau (OSTEP) – A fantastic, accessible deep dive into how operating systems actually work.
- Computer Systems: A Programmer’s Perspective (CS:APP) – The perfect guide for understanding how your C programs interact with hardware, memory, and the OS.
