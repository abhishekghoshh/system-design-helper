## Prerequisites

- SOLID
- Liskov Substituition Principle
- Design Patterns
    - Strategy Pattern
    - Observer Pattern
    - Decorator Pattern
    - Factory Pattern + Abstract Factory Pattern
    - Chain of Responsibility Design Pattern
    - Proxy Pattern
- [Concurrency Control in Distributed System](https://www.youtube.com/watch?v=D3XhDu--uoI)
